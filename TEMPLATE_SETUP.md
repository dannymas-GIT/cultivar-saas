# Template Setup for dannymas-GIT/saas-template-svelte-fastapi

1. Push this repo to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Initial template"
   git remote add origin https://github.com/dannymas-GIT/saas-template-svelte-fastapi.git
   git push -u origin main
   ```

2. On GitHub: Settings → General → check "Template repository"

3. The saas-scaffold skill will create new apps by:
   - Using "Use this template" on GitHub, or
   - Cloning and replacing cultivar-saas, Cultivar, CULTIVAR_SAAS placeholders
